# wets-cli

wets命令行工具

---

启动wets项目

```bash
$ wets start
```

编译wets项目

```bash
$ wets build
```

查看当前wets项目页面列表

```bash
$ wets page list
```

增加一个新页面

```bash
$ wets page add pageName
```

删除一个页面

```bash
$ wets page remove pageName
```

重命名页面

```bash
$ wets page rename oldPageName newPageName
```
